# number_classify
cnn 공부해보겠다고 선형대수학 수강했는데 넘파이로 하려니까 엄두가 안나서 일단 파이토치를 써보려 함

- 웹사이트: 리액트
- 모델: 파이토치
- API: FastAPI
시작시점에서 일단 아는게 많지 않아서 일단 이렇게만 구상함..
